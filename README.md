# Hackintosh-Catalina-10.15.7-EFI

EFI folder for MacOS Catalina 10.15.7

Hardware Specifications:  
AMD Ryzen 5 3600.  
Gigabyte b450m ds3h.     
MSI Radeon RX570 8GB.     
Corsair Vengeance LPX 8GB (1x8GB) DDR4 3200MHZ C16

Not Working:   
Sleep.  
Wifi and Bluetooth.  

Working:   
Everything else. 

Instructions:

Make a bootable usb. Refer the official guide:
https://dortania.github.io/OpenCore-Install-Guide/installer-guide/

Copy the contents of the EFI folder to the USB.

Connect ethernet and boot the usb for installation.
